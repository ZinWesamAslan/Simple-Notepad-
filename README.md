# Simple Notepad

A lightweight and efficient text editor developed using C# and Windows Forms. This application provides essential text editing functionalities and file management through a clean and intuitive user interface.

---

## Features

### File Menu
Includes core document management options:
* **Open:** Browse and open text files from your device.
* **Save:** Save changes to the currently active file.
* **Save As:** Save the text to a new file at a specified path.
* **Exit:** Close the application.

### Edit Menu
Standard tools for text manipulation:
* **Undo / Redo:** Revert or restore recent changes.
* **Cut, Copy, Paste:** Standard clipboard operations.
* **Delete:** Remove selected text.
* **Select All:** Highlight the entire text with a single click.

### Format Menu
* **Font:** Customize font family, size, and style (Bold, Italic, etc.) via a standard font dialog.
